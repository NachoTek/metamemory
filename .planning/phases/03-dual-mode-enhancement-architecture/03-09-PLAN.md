  <name>Task 2: Fix _replace_segment_in_display to handle enhanced segment replacement</name>
  <files>src/metamemory/widgets/floating_panels.py</files>
  <action>
    The `_replace_segment_in_display` method (around line 322-358) navigates to the segment using phrase_idx and segment_idx. However, when an enhanced segment comes in:
    
    1. It might come with a segment_index that doesn't match any existing segment in the current phrase
    2. The phrase tracking might be off because enhanced segments can arrive after new phrases have started
    
    The issue is that enhanced segments arrive ASYNCHRONOUSLY after the original transcription. The phrase/segment structure may have changed by then.
    
    Better approach: Use text-based matching to find and replace the enhanced segment.
    
    In `update_segment()` (around line 278-286), add special handling for enhanced segments:
    ```python
    if enhanced:
        # Enhanced segments: find and replace by text match
        self._replace_enhanced_segment(text, confidence, enhanced)
    elif segment_index < len(phrase.segments):
        # Update existing segment - REPLACE IN PLACE
        phrase.segments[segment_index] = text
        ...
    ```
    
    Add new method `_replace_enhanced_segment()`:
    ```python
    def _replace_enhanced_segment(self, new_text: str, confidence: int, enhanced: bool) -> None:
        """Replace a segment by finding matching text in recent phrases.
        
        Enhanced segments arrive asynchronously, so we find them by text match
        rather than by index.
        """
        # Search recent phrases for matching text (may have slight differences)
        # For simplicity, we can append enhanced text to current phrase
        # or search backwards for similar text
        
        # For now: append to current phrase with bold formatting
        # This ensures enhanced text is visible even if original position is lost
        if self.current_phrase_idx < 0:
            return
            
        cursor = self.text_edit.textCursor()
        cursor.movePosition(QTextCursor.MoveOperation.End)
        
        # Add space and enhanced text
        cursor.insertText(" ")
        
        color = self._get_confidence_color(confidence)
        fmt = QTextCharFormat()
        fmt.setForeground(QColor(color))
        fmt.setFontWeight(QFont.Weight.Bold)
        cursor.insertText(f"[ENHANCED] {new_text}", fmt)
        
        print(f"DEBUG Panel: Added enhanced segment: '{new_text}'")
    ```
    
    NOTE: A more sophisticated approach would store the original text when queued and match it back. But for immediate fix, the above ensures enhanced text is visible.
