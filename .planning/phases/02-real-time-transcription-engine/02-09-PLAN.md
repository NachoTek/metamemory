    2. Add SIGINT handler function before main() or in setup:
       ```python
       def setup_signal_handlers(app):
           def sigint_handler(signum, frame):
               print("\nReceived SIGINT, shutting down gracefully...")
               app.quit()
           
           signal.signal(signal.SIGINT, sigint_handler)
           
           # On Windows, signal handling needs a timer to work with Qt event loop
           import sys
           if sys.platform == 'win32':
               import win32api
               def win_handler(dwCtrlType):
                   if dwCtrlType == 0:  # CTRL_C_EVENT
                       app.quit()
                       return True
                   return False
               win32api.SetConsoleCtrlHandler(win_handler, True)
       ```
    
    3. Call `setup_signal_handlers(app)` in main() after QApplication creation
    
    Alternative simpler approach for cross-platform:
    ```python
    import signal
    signal.signal(signal.SIGINT, signal.SIG_DFL)
    ```
    This restores default Python behavior which works with Qt.
