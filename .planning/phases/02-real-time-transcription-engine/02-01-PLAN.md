<task type="auto">
  <name>Task 4: Create local agreement buffer for deduplication</name>
  <files>src/metamemory/transcription/local_agreement.py</files>
  <action>
    Create LocalAgreementBuffer class to prevent text flickering and duplication:
    
    - __init__(agreement_threshold: int = 2)
    - process_iteration(new_transcription: str) -> str - Returns newly committed text
    - get_committed() -> str - Get all committed text so far
    - get_pending() -> str - Get uncommitted (volatile) text
    - reset() - Clear all buffers
    
    LOGIC:
    - Keep buffer of recent transcriptions
    - Find common prefix between buffer and new_transcription
    - If common prefix == buffer: increment agreement count
    - Else: reset agreement count, update buffer to common prefix
    - When agreement count >= threshold: commit new text (buffer[len(committed):])
    - This prevents flickering - text only appears after 2+ consistent iterations
    
    ANTI-PATTERN TO AVOID: Don't emit text immediately - it flickers. Use local agreement.
    
    Reference RESEARCH.md Pattern 2 (line 103-144) and whisper_streaming implementation.
  </action>
  <verify>python -c "from metamemory.transcription.local_agreement import LocalAgreementBuffer; b = LocalAgreementBuffer(agreement_threshold=2); assert b.process_iteration('Hello world') == ''; assert b.process_iteration('Hello world') == ''; assert b.process_iteration('Hello world today') == 'Hello world'; print('Local agreement OK')"</verify>
  <done>LocalAgreementBuffer with agreement_threshold=2, commits text after N consistent iterations</done>
