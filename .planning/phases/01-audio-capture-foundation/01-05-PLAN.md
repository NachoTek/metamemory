Reorder stop() to stop sources BEFORE closing writer to prevent race condition:

Current order (lines 305-319):
1. Join consumer thread
2. Stop sources
3. Close writer

New order:
1. Stop sources (prevents new frames)
2. Join consumer thread (drains existing frames)
3. Close writer

Move this block from after line 311 to before line 305:
```python
# Stop all sources
for wrapper in self._sources:
    wrapper.stop()
```

This ensures no new frames are added while drain loop is consuming frames.
