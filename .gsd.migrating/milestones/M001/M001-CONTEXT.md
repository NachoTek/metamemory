# M001 Context

meetandread MVP — a Windows desktop widget application for silent, invisible audio transcription as cognitive augmentation infrastructure. Captures system audio and microphone input simultaneously, transcribes in real-time using local Whisper models with dual-mode parallel enhancement for high accuracy.

**In scope:** Audio capture (WASAPI), real-time transcription (faster-whisper), widget UI, confidence coding, FakeAudioModule testing.

**Out of scope:** Dual-mode enhancement, speaker identification, transcript management, advanced widget polish, system tray, macOS/Linux.

**Tech:** Python + PyQt6/PySide6, faster-whisper, WASAPI, all-local processing.
