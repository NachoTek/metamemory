# T07: Implement auto-scroll pause

**Slice:** S02 — **Milestone:** M001

## Description

Implement auto-scroll pause functionality: when user manually scrolls up, pause auto-scroll for 10 seconds to allow reading.

**Purpose:** Current behavior immediately fights user scroll by scrolling back down, making it impossible to read previous transcript content.
**Output:** Smart auto-scroll that detects manual scroll and pauses for 10 seconds before resuming.
